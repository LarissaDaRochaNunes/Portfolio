Projetos

Registro de plantações

Projeto desenvolvido em Python que se conecta a um banco de dados MySQL para realizar operações básicas de CRUD(Criar e Ler). O script permite ao usuário adicionar novos registros de plantações e listar todas as entradas existentes diretamente no terminal.

O principal desafio técnico foi implementar uma conexão de banco de dados segura. As credenciais (host, usuário e senha) são armazenadas em um arquivo config.py local, que é mantido fora do repositório através do .gitignore. Isso garante que nenhum dado sensível seja exposto publicamente no GitHub.
